# Estructura del Proyecto (Scaffolding) 

Para que el proyecto sea ordenado, seguiremos esta estructura básica. Aunque Expo nos da algunos archivos, nosotros crearemos la carpeta `src` para nuestro código.

```text
mi-app/
├── src/                # Carpeta principal de código
│   └── components/     # Todos nuestros componentes reutilizables
│       ├── BotonContador.js
│       ├── TarjetaInfo.js
│       └── ContenedorPrincipal.js
├── App.js              # Punto de entrada de la aplicación
├── package.json
└── ...
```
